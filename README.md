## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- Spring Initializr
- Maven
- Java 17
- Packing JAR
- Dependências:

  + Spring web
  + Lombok
  + Banco de dados H2
  + Spring JPA
  + Spring DevTools
 - Posteman
 - Git - Github

## 💻 Projeto

A finalidade do projeto é construir uma API Rest para realizar o cadastro de pessoas.
